sym_getInfinity

Get Symphony's infinity value

Calling Sequence

sym_getInfinity()

Description

Get the value that Symphony considers to be infinity. This is equal to 10^20.

Arguments

This function takes no arguments

Return value

Returns the infinity value of Symphony

Examples

sym_getInfinity()
